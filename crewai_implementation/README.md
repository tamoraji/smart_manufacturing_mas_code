# CrewAI Smart Manufacturing System
# This directory contains the CrewAI implementation of the smart manufacturing multi-agent system

## Directory Structure:
# agents/ - CrewAI agents (Data Intelligence, Preprocessing, Model, Optimization, Orchestration)
# tasks/ - CrewAI tasks with dependencies
# tools/ - Custom tools for manufacturing data analysis
# config/ - Configuration files for LLM providers and settings
# main_crewai.py - Main entry point for the CrewAI system

## Key Features:
# - Sequential task execution with dependencies
# - Memory-enabled agent collaboration
# - Human-in-the-loop support
# - Adaptive intelligence for model selection
# - Prescriptive maintenance recommendations
